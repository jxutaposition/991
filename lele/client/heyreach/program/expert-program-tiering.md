# Expert Program & Tiering

## Decisions Made
- Introducing tier levels for experts is critical to the program strategy
- MRR/churn scoring: points reset monthly (new MRR only); points earned are permanent — no full reset on churn, just decay. Source: log.md 2026-03-11
- Bojana's Expert Scoring System Notion doc (source of truth for tiers, rules, rewards): https://www.notion.so/heyreach/Expert-Scoring-System-tiers-rules-rewards-30609c3bcad3815c85aefd2f62212074
  - 4 tiers: Expert (0-29), Pro Expert (30-69), Elite Expert (70-119), Premium Partner (120+)
  - Points reset quarterly
  - Cash prizes — Quarterly top 3: $1,000 / $500 / $300; Yearly top 3: $5,000 / $3,000 / $1,000
  - Perks: customer intros, directory priority, early access (2 weeks), webinars/conferences, executive dinners, private Slack + founder access
  - Note: Notion API integration lacks access to this page (not shared with lele2.0 integration)
